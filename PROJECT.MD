# What is this project

Carimbo is a piece of software, designed to help people that use text and typing as a core part of their work. Carimbo acts as a toolbelt for quick inserting relevant text to have always on reach like people's full names, important cellphone numbers, ID numbers, a main website or really long urls, etc.

# Principles
1. Fast - Opening 'Carimbo' must feel lightweight and fast on all devices, even older, and in all OSes (Mac, Windows, Linux, Android...)
2. Intuitive - People using Carimbo should use it for everyday tasks and this must be really easy. Quick setup, install, create snippets and so on.
3. Accessible - High-contrast options, typography options etc to make it accessible for all and actually be even more valuable to people with bad vision, limited hand movement etc.

# Features
1. Snippets (core functionality) - All revolves around having relevant strings of texts at reach at all times, so if an user needs to fill forms frequently, add signatures to emails, copy and paste the same text all the time, they can rely on Carimbo to have faster work.
2. Clipboard - Clipboard history with intuitive UI so user never looses important CTRL-C's and CTRL-V's. Search, Pin, Folders and so on.

# Business-model
Carimbo is a free tool that can be used locally on any machine and the business-model is a cheap subscription (around $9 a year) for having cloud stored snippets and clipboard sync over user's devices.
